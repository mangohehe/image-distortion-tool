# Image Distortion Tool

A web-based SEM image augmentation tool built on Albumentations, specifically designed for semiconductor wafer defect detection workflows.

## Overview

This tool provides physics-based image distortions for training robust computer vision models on SEM (Scanning Electron Microscope) imagery. It bridges the gap between semiconductor domain expertise and machine learning workflows.

## Key Features

- **Quick Start Mode**: One-click presets for common SEM distortions
- **Custom Pipeline Builder**: Visual drag-and-drop interface for advanced users
- **Batch Processing**: Production-scale augmentation capabilities
- **Multiple Export Options**: Python code, JSON/YAML configs, Docker containers

## Target Users

- ML Engineers
- Semiconductor Domain Experts
- Data Scientists
- DevOps/Production Engineers
